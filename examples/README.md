# ZJE 本科毕业论文材料说明

> 浙江大学爱丁堡大学联合学院（ZJE）本科毕业论文（设计）材料索引。
> **本文件是整个 Skill 的行为规范。所有流程、顺序、阈值必须严格遵循，不按顺序操作可能导致材料作废。**

---

## 一、大前提：ZJE 写作顺序

ZJE 学生的写作顺序不同于传统本科论文：

```
Semester 1: IBMS10008 英文毕业论文（先写英文）
Semester 2: 将英文论文迁移为中文论文（再写中文）
        + Project Preview 文献综述和开题报告（英文原文）
```

**英文论文 → 中文论文迁移是本 Skill 的核心工作流。** 学生不是从中文学术写作开始，而是从已经完成的英文论文出发，按以下顺序完成各项材料。

### 第二组：填写附件2（Part 2 模板）

#### 步骤 2：填写附件2（第二部分模板）— [2/23]
- **描述**：将 Project Preview 的文献综述和开题报告**英文原文**填入附件2模板
- **参考文件**：
  - 附件2-第二部分模板-文献综述开题报告外文翻译.doc（空白模板）
  - 附件2-第二部分封面-格式说明.pdf（封面格式）
  - 第二部分正文编写说明-文献综述开题报告外文翻译规则.pdf
- **对应附件**：附件2
- **使用说明**：`第二部分正文编写说明.pdf` 明确了7条规则，包括：
  1. 正文全文中文字体为仿宋，英文字体为 Times New Roman，1.5倍行距
  2. 各标题和正文的字号大小参考附件2
  3. 各章之间统一分页
  4. 编排顺序：文献综述 → 开题报告 → 外文翻译 → 外文原文（**不可调换**）

- **✅ 通过阈值**：4 个部分全部填入且顺序正确，标题与英文论文主题一致
- **不通过后果**：第二部分内容不完整，查重后无法提交

## 五、ZJE 特殊性：英文论文 → 中文论文迁移

ZJE 学院要求学生**先完成英文毕业论文（IBMS10008 课程）**，再将英文论文迁移为中文论文提交教务。

### 迁移要点

1. **英文论文的章节标题直接翻译为中文**，填入附件8第一部分模板的各章章节标题位置；附件2中的标题仅供参考，内容不必相同。
2. **中文摘要**：将英文摘要翻译为中文，300–600字。
3. **英文摘要（Abstract）**：直接使用英文论文的 Abstract 内容。
4. **致谢（Acknowledgement）**：翻译英文致谢，中英文均可。
5. **第一部分正文**：将英文论文正文（Introduction / Methods / Results / Discussion / Conclusion）翻译为中文，字体仿宋小四号，1.5倍行距；参考文献和作者简历**不填入中文论文正文**（用于查重）。
6. **第二部分（文献综述 + 开题报告）**：来自 Project Preview 课程内容，可基于 Preview 稿件修改使其与 Dissertation 主题一致；无需翻译，直接使用英文原文填入附件2模板。
7. **外文翻译**：选一篇与毕设课题相关的**学术研究性论文**（非综述、非会议论文），自行翻译为中文，填入附件2第三部分；外文原文附在最后。
8. **所有材料中的 feedback**：Dissertation / Preview / Oral Presentation 的 feedback 均**原文粘贴，不需要翻译**。
