## 一个总原则：你先写英文，再迁移成中文

ZJE 和其他学院不一样——你先在 **IBMS10008** 课程写完**英文**论文，然后再把英文翻译成中文，填进教务处的模板。

```
英文论文（IBMS10008 已完成）          ← 你已经有的
    ↓ 翻译
中文内容（致谢/摘要/正文5章/参考文献）
    ↓ 填入模板
附件8（第一部分：论文主体）
    + 附件2（第二部分：文献综述+开题报告+外文翻译）
    + 附件1/3/6/9/10/11/17（各种表格）
    ↓
三个版本输出（查重版 / BB版 / 纸质版）
```

## 你需要做什么——按顺序来

### 第 1 步：看全局索引

打开 **`examples/中文论文材料顺序-全局索引.xlsx`**，把两大部分结构、三个版本的区别看明白。这是总地图。

### 第 2 步：完成 Project Preview 课程（截止：2026年2月28日）

拿到你的文献综述和开题报告（英文），后面会直接塞进附件2。

### 第 3 步：填附件2（文献综述 + 开题报告 + 外文翻译）

模板：`examples/附件2-第二部分模板-文献综述开题报告外文翻译.doc`

把 Preview 内容直接贴进去——**英文原文，不要翻译**。顺序不能换：
1. 文献综述
2. 开题报告
3. 外文翻译（找一篇学术研究论文，自己翻成中文）
4. 外文原文

格式要求：仿宋 / Times New Roman，1.5 倍行距，各章分页。

详细规则：`examples/第二部分正文编写说明-文献综述开题报告外文翻译规则.pdf`

### 第 4 步：填附件1（任务书）

模板：`examples/附件1-毕业论文任务书-空白表格.docx`
说明：`examples/附件1-毕业论文任务书.pdf`

从 Project Description 摘英文内容填入，日期填 2026年1月12日，控制在一页。

### 第 5 步：填附件11（毕业论文考核表）

说明：`examples/附件11-毕业论文考核表.pdf`

导师评语让导师写。Oral Presentation 的 feedback 从 Blackboard 复制粘贴（**不要翻译**）。

### 第 6 步：填附件3（文献综述考核表）

模板：`examples/附件3-文献综述和开题报告考核表-空白表格.docx`

把 Preview 的 feedback 从 Blackboard 复制粘贴（**不要翻译**）。

### 第 7 步：英文论文 → 中文论文（核心）

把英文论文以下章节翻译成中文，填入附件8模板：

| 英文章节 → | 中文章节 |
|------------|---------|
| Abstract | 中文摘要（300-600字） |
| Acknowledgement | 致谢 |
| Introduction | 引言 |
| Methods | 材料与方法 |
| Results | 结果 |
| Discussion | 讨论 |
| Conclusion | 结论 |

模板：`examples/附件8-第一部分模板-毕业论文主体.doc`

格式要求：
- 章标题：仿宋三号加黑
- 节标题：小三号仿宋加黑
- 正文：仿宋小四号，1.5倍行距
- 奇数页眉：论文名（居右）
- 偶数页眉："浙江大学本科生毕业论文（设计）"（居左）

### 第 8 步：整理参考文献

按 **GB/T 7714-2015** 格式。详细规则：`references/reference_rules.md`

### 第 9 步：填承诺书 + 简历 + 封面

承诺书：`examples/附件6-承诺书-空白表格.docx`
封面说明：`examples/附件8-大封面-格式说明.pdf`

### 第 10 步：查重（一次机会，≤ 10%）

**删掉**承诺书、参考文献、个人简历。用 Word 保存，**禁止 PDF**。

登录：http://check.cnki.net/pmlc/ → 学生入口
用户名：学号
密码：`zje`（小写）+ 身份证后 8 位

### 第 11 步：答辩 + 填答辩记录表

模板：`examples/附件10-现场答辩记录表-空白表格.doc`

### 第 12 步：专家评阅

把 Dissertation 的 feedback 粘贴到：`examples/附件9-专家评阅意见表-空白表格.doc`

### 第 13 步：生成三个版本

| 版本 | 包含 |
|------|------|
| **查重电子版** | 封面 + 致谢 + 摘要 + 目录 + 正文（去掉参考文献和简历） |
| **BB提交电子版** | 全部正文 + 所有附件，不含任务书/考核表等内部文件 |
| **纸质存档版** | 全部 20 项材料按顺序装订，浅黄色皮纹纸，打印 1 份 |

打印：海宁校区校内打印店，挂账"ZJE + 你的名字 + 毕业论文"

## 红线（绝对不能碰）

| 规则 | 后果 |
|------|------|
| **查重率 > 10%** | 影响毕业 |
| **承诺书不签字** | 材料退回 |

| **查重版没删参考文献** | 重复率偏高 |
